# Simple Text Display Webpage

This project is a simple text display webpage built using HTML and JavaScript. The webpage dynamically displays text using JavaScript, demonstrating basic DOM manipulation.

## Features
- Simple text display
- JavaScript DOM manipulation
- Beginner-friendly project
- Clean and simple UI

## Technologies Used
- HTML
- JavaScript

## How to Run
1. Download or clone the repo.
2. run index.html file.
